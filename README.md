<!-- GETTING STARTED -->
## Getting Started

### Prerequisites


* npm
  ```sh
  npm install npm@latest -g
  ```
  
  * npm
  ```sh
  npm install gulp-sass sass --save-dev
  ```
  

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build with gulp
   ```sh
   npx gulp
   ```
